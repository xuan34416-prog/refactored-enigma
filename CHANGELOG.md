## 📌 版本说明

### 版本号规则
- **主版本号**：重大功能更新（不兼容的 API 变更）
- **次版本号**：新功能添加（向后兼容）
- **修订号**：Bug 修复和小改进

### 数据保留政策
- ✅ **覆盖安装**：数据完全保留
- ✅ **同签名更新**：数据完全保留
- ❌ **卸载重装**：数据会丢失

### 更新方式
1. **APK 覆盖安装**（推荐）
   - 下载最新 APK
   - 直接安装到手机
   - 选择"替换现有应用"
   - 数据保留 ✓

2. **PWA 自动更新**
   - Service Worker 自动检测更新
   - 下次启动时应用新版本
   - 数据保留 ✓
